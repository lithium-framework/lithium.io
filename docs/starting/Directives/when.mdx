# When

La directive `when` de Lithium est utilisée pour gérer l'affichage conditionnel de contenu en fonction de l'évaluation d'une expression booléenne. Elle est particulièrement utile pour afficher différents contenus en fonction de l'état des données ou des conditions.

### Fonctionnalité

- **Affichage Conditionnel** : `when` permet de rendre des parties spécifiques du template uniquement lorsque certaines conditions sont remplies. Vous pouvez spécifier différents contenus à afficher pour les cas vrais ou faux de la condition.
- **Réactivité** : La directive `when` réagit automatiquement aux changements des conditions définies, ce qui permet de mettre à jour le rendu en fonction de l'état des données ou des conditions.

### Concept Clé

La directive `when` est essentielle pour la gestion des états conditionnels dans les templates Lithium. Elle simplifie le rendu conditionnel en permettant de spécifier directement dans le template quel contenu afficher pour chaque condition.
