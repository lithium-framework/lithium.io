# AsyncAppend

La directive `asyncAppend` est utilisée pour gérer les promesses dans les templates Lithium. Elle permet de rendre le résultat d'une promesse de manière asynchrone dans le DOM, ce qui est particulièrement utile pour afficher des données provenant de requêtes réseau ou d'autres opérations asynchrones.

### Fonctionnalité

- **Gestion asynchrone** : La directive `asyncAppend` attend qu'une promesse soit résolue et met à jour le DOM avec le résultat de cette promesse.
- **Réactivité** : Lorsqu'une nouvelle valeur est disponible, elle est automatiquement rendue dans le template.
