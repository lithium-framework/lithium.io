# Repeat

La directive `repeat` de Lithium est utilisée pour générer des éléments répétitifs dans un template en fonction d'une collection de données. Elle facilite la création dynamique de listes d'éléments basées sur des données fournies, en rendant chaque élément de manière efficace et réactive.

### Fonctionnalité

- **Répétition Dynamique** : `repeat` permet de générer plusieurs instances d'un template en fonction des éléments d'une collection. Cela est particulièrement utile pour afficher des listes ou des tableaux de manière dynamique, où chaque élément de la collection est rendu par un template spécifique.
- **Performance Optimisée** : La directive est conçue pour être efficace en termes de performance, en minimisant le nombre de mises à jour nécessaires et en optimisant le rendu des éléments répétés.
- **Flexibilité** : Vous pouvez utiliser `repeat` pour créer des listes d'éléments basées sur n'importe quelle collection, en passant un template qui définit comment chaque élément doit être rendu.

### Concept Clé

La directive `repeat` est un outil puissant pour générer des listes ou des tableaux d'éléments dans Lithium. Elle permet de gérer efficacement des collections de données en rendant chaque élément avec un template défini, ce qui simplifie la création dynamique de contenu répétitif.
